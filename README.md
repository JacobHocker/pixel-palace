# Pixel Palace
An enjoyable arcade full of the most known & fun games. 

### [Visit Pixel Palace](https://pixel-palace.vercel.app/)

### Built with 
-   React
- CSS
- Styled Components
- Framer Motion
- TypeScript

## Themes
> Pick your theme of four to customize  your Pixel Palace experience.
- Retro
- Cyber
- Classic
- Modern
